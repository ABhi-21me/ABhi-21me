## 🛠️ what I've shipped

The strongest evidence for the cloud/security side of what I do is **CloudGrid** — a CNAPP (Cloud Native Application Protection Platform) dashboard on AWS that uses **Amazon Bedrock (Claude 3 Haiku)** for automated threat detection and remediation. Tested it against SQL injection, XSS, and SSH brute-force scenarios. Redesigned the whole UI around a dark terminal-ops aesthetic — React, Vite, TypeScript, Tailwind, Recharts, Zustand.

The other 30% is rapid, AI-assisted product builds — I use tools like Lovable, Cursor, and direct Claude API integration to go from idea to working app fast, and I'm upfront that this is the workflow, not something to hide:

- **SafeRoute** — Flutter + Firebase women's-safety app with BLE follower detection, presented at IntrusionX SE hackathon
- **BoomRoom** — multi-device synced party speaker, WebRTC + Socket.io
- **BrewPulse** — React/Node.js/MySQL HMI dashboard for an industrial liquor-dispensing system, built under faculty guidance
- **Paisa Guru** — Hinglish AI expense manager with conversational logging
- **GhostLink / GhostChat** — AES-256-GCM encrypted chat (Web Crypto API) and ephemeral passcode-protected chat with hover-reveal text
- A web-based strength-training tracker with a bento-grid dashboard, since rebuilt as a Flutter + Riverpod + SQLite mobile spec

Full list lives in the repos tab — I'd rather you check the code than take my word for it.
